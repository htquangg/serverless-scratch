# Serverless Framework From Scratch

## Prerequisites

You'll need [Node.js](https://nodejs.org/en/), [npm](https://www.npmjs.com/) installed.

## Getting Started

```bash
git clone https://github.com/htquangg/serverless-scratch.git
cd serverless-scratch
npm ci && npm run dev 
```
